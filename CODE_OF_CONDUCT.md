# Code of Conduct

This project uses a simple standard: be direct, kind, and useful.

Disagreement is welcome. Personal attacks, harassment, slurs, or repeated bad
faith behavior are not.

Assume experimental results can be messy. Critique claims, code, tests, and docs
with enough detail that someone else can improve them.

Maintainers may remove comments, issues, pull requests, or participation that
make the project harder for reasonable people to work on.
